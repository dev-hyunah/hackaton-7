# 운임관리

## ‘노선 및 일자 설정 영역

- 달력으로 날짜를 변경하면 아래에 표출되는 날짜는 달력에서 변경된 날짜를 기준으로 -3d, +3d를 좌우로 배치해서 일주일이 나오도록 수정 필요

## ‘운항현황’ 영역

- ‘현재가’는 제거필요
- ‘AI 추천’은 적절한 문구로 생성 필요(유지 / 가격을 올리세요 / 가격을 내리세요 등 적절한 문구를 찾아줘.)
- ‘상태’에 위험이 적절한 것인지 다시 확인 및 수정 필요(‘매진임박’ 과 같은 용어가 적절해 보여.)

## ‘좌석 등급별 운임 관리’ 영역

- 전체 공급되는 좌석의 수는 고정된 상태에서 AI가 수익 극대화가 될 수 있도록 상황에 따라서 좌석 등급별 좌석 수를 적절하게 추천해주는 기능 추가 필요 (현재 가격 추천 기능과 유사한 기능)
- ‘상세보기’ 팝업에서 ‘거부’ 시 AI 추천을 거부한 것이므로 상세보기 버튼이 비활성화 되고, 상세보기를 포함한 AI추천 영역이 비활성 표시(색상 변경 등의 효과) 필요
- Closed 된 등급에 잔여 좌석이 존재할 경우, 다른 등급으로 잔여 좌석을 자동으로 이관하는 기능 추가 필요

## 'Profit Analysis' 영역

- '인벤토리 실시간 통제 확정' 버튼 클릭 시 백엔드에 변경사항을 저장하는 onClick 핸들러 구현 필요

## 'AI 전략 분석 요청' 영역

- 'AI 전략 분석 시작' 버튼 클릭 시, AI 를 통해 분석된 실제 응답 필요. 현재 응답이 하드코딩 되어있음
- [hyunah] 'AI 전략 분석 시작' 버튼 클릭 시 AI model을 실제로 호출하여 응답을 받아 표시해야 함
