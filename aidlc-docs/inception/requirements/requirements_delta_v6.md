# 전체

- 현재는 운항하고 있지 않은 노선도 선택 가능한데 실제 대한항공 국내선 운항 노선대로 선택 가능 노선을 변경해줘
- 금액 표기시 숫자 읽기 쉽게 1,000 단위마다 콤마를 삽입해줘
- 모바일에서 확인했을 때 화면 및 텍스트 영역이 깨지지 않도록 반응형으로 구현해줘

# 대시보드

- 노선 변경해도 수익 추이 데이터가 변경되지 않아 그래프 변동이 없는데 노선 변경시 수익 추이 데이터가 변경되도록 노선별 수익 데이터 변경해줘
- 1일, 3일, 7일, 10일 일자 필터링 변경시 '평균 Load Factor', 'AI 승인 대기' 데이터 변동이 없는데 일자 필터링 변동시 데이터 변동되도록 데이터 변경해줘
- '최근 1일 수익' 금액 표기시 1,000 단위마다 콤마 삽입해줘

# 운임 관리

- 새로고침 클릭시 정상적으로 새로고침 잘 되는지 새로고침 버튼 옆에 마지막 새로고침 시간 표출해줘 (초 단위까지 표출)
- 새로고침 버튼 우측 항공편명과 출발 시간 표출은 제거해줘
- 좌측 운항편 판매현황 하단 (ex. B737-900ER (200석) · GMP-CJU) 데이터 표출도 제거해줘
- 특정 운항편 클릭하여 이동한 상세 페이지에서 상단 '운항 목록' 텍스트 우측에 현재는 예를 들어 'KE1207 09:30(오전) B737-800'와 같이 표출되는데 DDMMM 날짜 형태로 날짜도 추가하여 표출되도록 수정해줘
- 현재 좌석 공급석 수정시 UI가 작고 눈에 띄지 않는데 사용자친화적으로 수정하기 편리하도록 UI 수정해줘
- 현재 L/F progress bar 굵기가 얇아서 헷갈리는데 지금보다 2배 더 굵게 수정해서 가시성을 높여줘
- 좌석 공급석 수정시 숫자 3자리가 넘어가는 경우 input 영역 안의 입력된 숫자가 잘리는데 이러한 경우 input 영역 너비를 동적으로 늘어나도록 수정해줘
- 공급석 input 표출 시 숫자 3자리 초과해도 숫자가 잘리지 않도록 input 영역 너비를 자릿수에 맞게 충분히 확보해줘

# 시뮬레이터

- 노선 선택에서 '국내선 전체' 옵션도 추가해주고, 기본 옵션은 '국내선 전체' 옵션으로 선택해줘
- 노선 선택 변경시 '예수 수익', '예상 Load Factor', '예상 일일 예약' 등 데이터가 변동되도록 데이터 변경해줘
- '예상 수익 (일평균)', '예상 일일 예약' 데이터 표시지 1,000 단위마다 콤마를 삽입해줘
- 현재 '국내선 전체' 옵션만 선택이 되는데 초기값 선택이지 다른 노선도 선택 가능하도록 수정되어야 해
- 노선 선택 변경 시 슬라이더(유가 변동, 환율 변동, 자사 운임 조정) 값과 시뮬레이션 결과가 초기화되지 않도록 수정해줘
- '초기화' 버튼 클릭 시에만 노선 선택 포함 모든 값이 초기화되도록 수정해줘 (초기화 시 노선은 '국내선 전체'로 복귀)
